FizzBuzz: A coding project

When it counts you fizzbuzz. A Javascript coding exercise using loops and DOM manipulation. Write a programme that prints the numbers from 1 to 100 and for multiples of '3' displays 'Fizz' instead of the number and for the multiples of '5' displays 'Buzz'. If the number is a multiple of both, it displays 'FizzBuzz'. The programme has the flexibility to change the numbers to any other numbers, instead of just 3 and 5
